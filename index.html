About This Project

This is a basic music player inspired by Spotify, built using HTML, CSS, and JavaScript. It features a playlist, play/pause controls, progress tracking, and a dynamic UI that highlights the currently playing song. Perfect for learning DOM manipulation, event handling, and the HTML5 Audio API.
